# Kurulum 
- İlk olarak CMD'ye `npm install` yazın.
- config.js dosyasına girip token ve mongodb alanını doldurun.
- doldurduktan sonra CMD'ye `npm start` veya `note bot.js` yazın ve sorunsuz aktif <3.
